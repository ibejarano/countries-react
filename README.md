> App React para buscar informacion basica de un pais

## Features

- Buscar paises
- Ver informacion sobre cada pais
- Ver paises de las fronteras de cada pais

---

## Installation

Software necesario:

- node
- npm
- git

### Setup

```shell
$ npm install
```

Iniciar en modo development

```shell
$ npm run start
```
